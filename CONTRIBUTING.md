# Contributing

Contributions are welcome! Here's how to get started:

1. Fork the repository and clone your fork
2. Set up the development environment (see [DEVELOPMENT.md](DEVELOPMENT.md))
3. Create a feature branch (`git checkout -b fix/your-bugfix-name`)
4. Make your changes and add tests in `tests/unit/`
5. Run `pytest tests/unit/ -v` to verify
6. Commit, push, and open a Pull Request

## Reporting Bugs

[Open an issue](https://github.com/d-krupke/skbuild-conan/issues/new) with:

- Steps to reproduce
- Expected vs actual behavior
- Your environment (OS, Python version, Conan version)
- Logs from `SKBUILD_CONAN_LOG_LEVEL=debug pip install ...`

## Questions?

Open a [GitHub issue](https://github.com/d-krupke/skbuild-conan/issues/new).
